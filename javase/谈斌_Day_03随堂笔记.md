#### 逻辑运算符

```java
// 在Java中不允许 1 <= a <= 5 这样表示a的范围,必须要使用逻辑运算符

分类:
	并且 : &  -> 与

    非 : ! -> 非
        
拓展的逻辑运算符:
	逻辑双与 : &&

逻辑运算符是用来链接关系表达式的,逻辑运算符的结果也一定是 boolean 类型

	逻辑与: 有false则为false -> 并且
```

#### 逻辑双与和逻辑双或

```java
逻辑双与和逻辑双或: 都实现了短路效果
    
运算规则:
	逻辑双与: 有false则为false -> 并且
    逻辑双或: 有true则为true -> 或者
        
短路效果:
	逻辑双与: 左边为false,右边不执行
    逻辑双或: 左边为true,右边不执行    
        
//要是用逻辑与和逻辑或 优先使用 逻辑双与和逻辑双或 ,提高了代码的运算效率        
```

##### if第一种格式

```java
格式:
	//代码1;
	if(任何结果为boolean类型的式子){
         语句体;
    }
	//代码2;
执行流程:
	1. 计算小括号内式子的结果
    2. 若结果是true,执行大括号内的语句体
    3. 若结果是false,不会执行大括号内的语句体,而是执行if结构后面的代码 //if结构不会影响程序往下执行  
        
if的第一种格式是对 1 种情况的判断. (只对小括号内为true的时候做操作)
        
伪代码:
	if(小丽来不来){
        去小树林走走; 
    }

//程序员买西瓜 
	int num = 1;//购买西瓜的数量
	if(看到西红柿){
        num = 2;
    }
```

##### if第二种格式

```java
格式:
	//代码1;
	if(任何结果为boolean类型的式子){
         语句体1;
    }else{
         语句体2;
    }
	//代码2;

执行流程:
	1. 计算小括号内式子的结果
    2. 若结果是true,执行语句体1
    3. 若结果是false,执行语句体2 
    4. 当语句体执行完毕,执行if结构后的代码
        
伪代码:
	if(小丽来不来){
        去小树林走走; 
    }else{
        约小丽闺蜜小红;
    }

if的第一种格式是对 2 种情况的判断.
```

##### if第三种格式(标准)

````java
格式:
	//代码1;
	if(任何结果为boolean类型的式子1){
         语句体1;
    }else if(任何结果为boolean类型的式子2){
         语句体2;
    }else if(任何结果为boolean类型的式子3){
         语句体3;
    }
	..这里可以无限加else if..	
	else{
         语句体n + 1;
    }
	//代码2;	

执行流程:
	1. 判断关系表达式1的结果,若结果是true,执行语句体1,并结束if语句
    2. 若关系表达式1的结果是false,判断关系表达式2的结果,若结果是true,执行语句体2,并结束if语句  
    3. 若关系表达式2的结果是false,判断关系表达式3的结果,若结果是true,执行语句体3,并结束if语句   
        .....
    4. 若所有的关系表达式结果都为false,若有else,执行语句体n+1;   

伪代码:
	if(小丽来不来){
        去小树林走走;
    }else if(小丽的男朋友来了?){
        跑;
    }else if(小丽的妈妈来了?){
        请小丽妈妈吃饭;
    }else{
        约小丽闺蜜小红!
    }
````

#### switch结构(理解)

```java
格式:
	switch(表达式){
        case 值1:
            语句体1;
            break;
        case 值2:
            语句体2;
            break;
        case 值3:
            语句体3;
            break;
           ..可以有很多case..
        default:    
            语句体n+1;
            break;
    }

格式解释:
	switch : 关键字-> switch分支结构
    (表达式) : 表达式的内容的类型只能是 -> byte,short,char,int,字符串,enum(枚举)  //一般是变量
        作用: 表达式的值要和case后面的值做 == 比较;
	case : 事件/情况  //有多少个case就代表有多少种情况
    值 : 
		1. 要求和表达式的类型一致
        2. 一般是常量或者是枚举   
        3. 值与值之间不能重复    
        作用: 等待表达式的结果来比较的
    语句体: 可以是多句代码;//一般不会放很多代码
		//当表达式的结果和当前case后的值相等就需要执行case中的语句体;
	break : 打断,中断 
        作用: 用来结束switch语句的
    default : 默认的,缺省的
        作用: 类似以if中的else, 当switch结构中所有的case都不匹配,执行default中的语句体n+1;

执行流程:
	1. 计算表达式的结果
    2. 从上至下依次和case后的值进行==比较
    	 	若相等,执行当前case中的语句体,遇到break结束switch语句;
			若不相等,继续下一个case比较,直到遇到相等的case值进入当前case;
	3. 当所有的case值和表达式的结果都不匹配,若有default语句,执行default中的语句体n+1;
```

#### 循环结构

```java
循环结构的组成:
	1. 初始化语句 : 定义变量
			作用: 变量控制循环执行
    2. 判断条件语句 : 结果为boolean类型的式子
        	作用: 为true,循环执行. 为false,循环结束
    3. 控制条件语句(步进表达式) : 初始化语句定义的变量的变化!变化方向往不满足条件的方向前进
            作用: 为了让循环停下来
    4. 循环体语句   : 重复执行的内容 
```

#### for循环

```java
for循环是最为常用的循环结构之一!
    
格式:
	for(初始化语句1;判断条件语句2;控制条件语句(步进表达式)3){
        循环体语句4;
    }

执行流程:
	1. 执行初始化语句
    2. 执行判断条件语句
        若结果是false,立即停止循环
    3. 若判断条件语句的结果是true,执行循环体语句
    4. 执行步进表达式
    5. 从步骤2开始重复执行.......
        
执行流程代码:
	1 --> 2 -true-> 4 --> 3 --> 2 -true-> 4 --> 3 .....
            -false-> 结束			 -false-> 结束
```
